## Getting Help

### Common Issues

#### GitHub Token Issues
**Problem**: Repository operations failing  
**Solution**: Verify token has `repo` permission

#### AI API Errors  
**Problem**: Code analysis not working  
**Solution**: Check Gemini API key is valid

#### Firebase Connection Issues
**Problem**: Evolution history not saving  
**Solution**: Verify Firebase configuration
