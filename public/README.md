# Public Directory

Static assets that should be served from the root URL.

## Contents

This directory typically contains:
- `favicon.ico` - Site favicon
- `robots.txt` - SEO robots file (already in root)
- `sitemap.xml` - SEO sitemap (already in root)
- Images, fonts, and other static files

## Usage

For static file hosting, place files here. For GitHub Pages, these are served from root level.

## Note

Currently using root-level files. This directory is for future static asset organization.
