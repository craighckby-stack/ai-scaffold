# Well-Known Directory

Public information accessible at /.well-known/ endpoint.

## Standard Files

- `security.txt` - Security policy contact information
- `robots.txt` - Already in root
- `humans.txt` - Already in root
- `change-password.txt` - Password change URL (not applicable)
- `keybase.txt` - Keybase verification (optional)

## RFC 5785

This directory follows RFC 5785 for well-known URIs.
